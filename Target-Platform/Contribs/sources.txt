This file indicates where each of the zip files in this directory came from and what uses them:

Zip files downloadable directly from the web and unmodified:
 - NOTE: the contents of some of these zips are not in an /eclipse directory, but the build script catches that and adjusts
* eclipse-test-framework-4.7.3.zip -> http://download.eclipse.org/eclipse/downloads/drops4/R-4.7.3-201803010715/download.php?dropFile=eclipse-test-framework-4.7.3.zip
 - used for the CC junit test projects
* org.eclipse.pde.source-4.7.3.zip -> http://download.eclipse.org/eclipse/downloads/drops4/R-4.7.3-201803010715/download.php?dropFile=org.eclipse.pde.source-4.7.3.zip
* GEF3-Update-4.0.0.zip -> http://www.eclipse.org/gef/downloads/index.php under GEF-Legacy 4.0.0 (2016/06/06) -> GEF-Legacy Update Site
   -> http://www.eclipse.org/gef/download.php?file=/tools/gef/downloads/drops/legacy/4.0.0/R201606061308/GEF3-Update-4.0.0.zip
      -> http://www.eclipse.org/downloads/download.php?file=/tools/gef/downloads/drops/legacy/4.0.0/R201606061308/GEF3-Update-4.0.0.zip
 - used simply to get source attachements, for things like org.eclipse.gef_3.11.0
* EMF-Updates-2.13.zip -> https://www.eclipse.org/downloads/download.php?file=/modeling/emf/emf/builds/release/2.13/EMF-Updates-2.13.zip
   -> http://download.eclipse.org/modeling/emf/emf/builds/release/2.13/index.html
 - used for source attachements for emf development
Zip files created by downloading features from an update site, then packaging up the features & plugins found in the eclipse install into a zip archive:
* quickimage_1.0.3.2.zip -> http://psnet.nu/eclipse/updates

* org.eclipse.nebula-1.3.0.zip -> https://eclipse.org/nebula/downloads.php (starting from repository-nebula-latest.zip)
 - XYPlot widget
