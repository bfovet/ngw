Instructions on using the Compare Current Editor With... (diff) utility

In the "Select a file to compare" window, a list of active tabs is shown. Tabs that cannot be compared with are not listed. You may select one of the active tabs to compare, or press the "Choose external file" button to bring up a file dialog and select a file from the file system to compare the current editor with.


Be sure to save any changes to the current editor before comparing. The compare editor will either replace a currently open compare editor or open in a new tab. See Window->Preferences->General->Compare/Patch->Compare (Diff) to update these preferences.
