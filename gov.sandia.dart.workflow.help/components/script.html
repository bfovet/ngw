NGW Component Documentation - script

script

Description

Runs a user-supplied script in-process.

Properties

customCode - the script to execute.

Input Ports

x - the single default input. You can add more inputs if desired. The values at all the inputs will be available to the script as global variables with the same names as the ports.

Output Ports

f - the single output. Your script should set the global variable "f" to produce a value at this output.

Notes

Currently implemeted in terms of JavaScript. Will be moved to Python at some point.
