NGW Component Documentation - arrayElement

arrayElement

Description

Given an array input, extract a single element from that array.

Properties

index - an integer, the zero-based index of an element to return from the array. The value -1 can be supplied; in this case the last element of the array will be returned.

Input Ports

array - an array value, such as a response output from a nestedWorkflow node

index - an integer, the zero-based index of an element to return from the array. The value -1 can be supplied; in this case the last element of the array will be returned. As is generally the case, a value supplied at this port will override a value from the property of the same name.

Output Ports

element - the desired element of the array
