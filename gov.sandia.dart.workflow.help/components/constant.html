NGW Component Documentation - constant

constant

Description

Produce a given constant value on the lone output.

Properties

value - the value of the constant.

Input Ports

(none)

Output Ports

f - the value is produced on this output

Notes

Value may be character or numeric data. A constant node is similar to a parameter node, but it isn't "visible" outside the workflow. It won't appear as an input parameter to this workflow if it is nested inside another workflow.
