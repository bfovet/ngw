NGW Component Documentation - fileChooser

fileChooser

Description

Allows a user to browse to and select a file.

Properties

initialPath - the directory at which the browser should start. If not specified the workflow home directory is used.

title - a title for the browser dialog.

fileFilters - one or more allowed file patterns; i.e., "*.txt;*.dat".

Input Ports

x - a pure trigger; data is ignored

Output Ports

f - if the user chooses a file, an absolute filesystem path to the chosen file.

canceled - if the user cancels the operation, the string "canceled" is sent here.

Notes

If the workflow is re-executed with a populated work directory, will remember the previously selected file and try to reselect it.
