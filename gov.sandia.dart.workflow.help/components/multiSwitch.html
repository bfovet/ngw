NGW Component Documentation - multiSwitch

multiSwitch

Description

Send a value read from an input to one of many output ports chosen based on a value read from a second input.

Properties

selector - the name of an output port

passThrough - any data

Input Ports

selector - the name of an output port

passThrough - any data

Output Ports

_else - the default port for output if the selector value doesn't match any user-created ports.

Notes

To use this node, manually add some number of output ports. Supply the name of one of these ports to the "selector" input or property. The value in the "passThrough" port or property will be sent to that named output when the node fires. If the value from the selector does not name an output port, the data will be sent to the _else port.
