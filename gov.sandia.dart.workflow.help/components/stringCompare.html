NGW Component Documentation - stringCompare

stringCompare

Description

Compares an input string to another string. Returns true if the two match completely; returns false if they do not

Properties

compareString - the string to be compared with the input

Input Ports

x - the input string that is to be compared with the compareString defined in Properties

Output Ports

true - if the two strings match, this output port will be followed (and gets the value of the input string)

false - if the two strings don't match, this output port will be followed (and gets the value of the input string)

Notes
