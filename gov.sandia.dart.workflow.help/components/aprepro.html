NGW Component Documentation - aprepro

aprepro

Description

Runs the aprepro processor on a file, stores the result in a new file, and passes along the path to this new file. The current values of the workflow's parameters are available as variables in the aprepro environment.

Properties

templateFile - a local file, the file to process using aprepro..

commentChar (Optional) - a comment character (or string) appropriate to the file being processed. If unspecified, defaults to "#".

outputFile (Optional) - The desired name for the output file. If unspecified, for filename.ext defaults to filename.aprepro.ext .

Input Ports

(None)

Output Ports

outputFile - absolute path to the processed file

Notes

Requires configuration: the environment variable APREPRO_PATH should exist in the workflow server environment, and point to the aprepro executable.
