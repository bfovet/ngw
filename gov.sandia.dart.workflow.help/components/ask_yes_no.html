NGW Component Documentation - ask_yes_no

ask_yes_no

Description

Displays a dialog that asks a yes or no question. Depending on the answer, data from the input "x" will be sent to either the "yes" or "no" outputs.

Properties

question - a yes or no question

Input Ports

x - input data, which will be directed to one of the outputs depending on the user's answer to the question

Output Ports

yes - input from x will be sent to this output if the user response to the prompt is "yes"

no - input from x will be sent to this output if the user response to the prompt is "no"

Notes

Obviously this node is intended for interactive workflows only, and may not be available in standalone NGW installations.
