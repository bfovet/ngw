NGW Component Documentation - downloadFile

downloadFile

Description

Downloads a file

Properties

hostname - the machine to download the file from

localFile - optional; a local path for the downloaded file

remoteFile - path of the remote file to be downloaded

remotePath - the working directory to use on the remote machine

Input Ports

localFile - a local path for the downloaded file/p>

Output Ports

file - the absolute path of the downloaded file.
