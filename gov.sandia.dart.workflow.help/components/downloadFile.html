NGW Component Documentation - downloadFile

downloadFile

Description

Downloads a file either locally or remotely

Properties

hostname - the location of the file to be downloaded

localFile - optional; determines the file to be downloaded

remoteFile - determines the file to be downloaded

remotePath - If downloading a file remotely, specifies the working directory to use on the remote machine

Input Ports

trigger - any node. It will initiate the download

Output Ports

outputFile - any port that could use the file such as fromFile

Notes

The localFile, if given, could be an absolute path. If it's not, then it's relative to the workflow working directory
