NGW Component Documentation - Note

Note

Description

A text box that remains on the workflow space; can be used to describe the workflow or provide instructions, for example. A note does nothing when the workflow runs; it is essentially just decoration in the editor.

Notes

You can change the color of a note, or you can turn off background and outline drawing altogether.

You can stretch out a note and put active workflow nodes on top of it, so the note serves to visually group those nodes together.
