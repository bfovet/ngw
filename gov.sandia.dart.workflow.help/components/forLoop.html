NGW Component Documentation - forLoop

forLoop

Description

Used in conjunction with the endloop node to create a for loop within the workflow

Properties

start - the number at which the for loop starts

step - the step size

end - the number at which the for loop ends; if the step causes a change that exceeds this value, the for loop will terminate

Input Ports

x - a previous value to be used within the loop

_LBEGIN_ - once all nodes have been included in the loop, connect the last one to an endloop node and then connect the _LEND_ of the endloop node to this port

start - the number at which the loop starts

step - the step size

end - the number at which the for loop ends; if the step causes a change that exceeds this value, the for loop will terminate

Output Ports

f - begins the loop

Notes

If start, step, and end are specified in both Properties and the Input Ports, the Properties values will take precedence
