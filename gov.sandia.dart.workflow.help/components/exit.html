NGW Component Documentation - exit

exit

Description

Prints a message to stdout and halts the executing workflow by canceling execution.

Properties

formatString - a Java format string. See https://docs.oracle.com/javase/tutorial/java/data/numberformat.html.

Input Ports

x - triggers the exit. The value can be embedded in the printed output via format conversion.

Output Ports

None.
