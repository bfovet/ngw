NGW Component Documentation - templateSubstitution

templateSubstitution

Description

For each property and/or input port added to this node, replace all instances of the name of the added property or inputs port that are found within templateString with the value supplied for the given property or input port.

Properties

templateString
the string on which subsitution is to be performed
all others
the value to be substituted for the name of the given property (or input port)

Input Ports

See Properties above.

Output Ports

instantiatedTemplate
the string after substitution
