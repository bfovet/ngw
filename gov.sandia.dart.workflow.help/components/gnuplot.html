NGW Component Documentation - gnuplot

gnuplot

Description

Generates a basic x-y chart in gnuplot and generates an image file.

Properties

imageFile - the name of the file to emit.

commands - any number of gnuplot commands to execute after the data is loaded but before the plot is generated.

Input Ports

x - an array, the independent variable for the plot

y - an array, the dependent variable for the plot

Output Ports

(none)

Notes

set title "My plot"set xlabel "time"set ylabel "voltage"set grid
