NGW Component Documentation - endloop

endloop

Description

Used in conjunction with a forLoop node to create a for loop within the workflow. Signifies that the end of the loop has been reached. The node will either redirect to the beginning of the loop or exit the loop

Properties

(None)

Input Ports

x - any value or string that was generated within the for loop

Output Ports

f - the end result of the for loop. This port will only be used once the for loop has completed all of its iterations

_LEND_ - this port redirects to the beginning of the loop and should be connected to the _LBEGIN_ port of the forLoop node
