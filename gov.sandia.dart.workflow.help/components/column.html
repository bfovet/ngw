NGW Component Documentation - column

column

Description

Extracts a column of data from a tabular data file.

Properties

columnIndex - the 1-based index of the column to extract (left to right)

headerLines - the number of lines to discard at the beginning of the file. Defaults to 1

separator - a Java regular expression matching the separators between columns. Defaults to "\\s+" (any white space.)

Input Ports

Note: you should specify only one of the following two inputs.

data - raw tabular data.

fileName - the name of a file containing tabular data.

Output Ports

stdout - the requested column of data.
