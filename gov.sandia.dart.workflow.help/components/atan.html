NGW Component Documentation - atan

atan

Description

Computes the arctan of the input, x, i.e. the angle, f, between -π⁄2 and π⁄2 in radians such that tan(f) = x

Properties

(None)

Input Ports

x - a number (the ratio between the lengths of the opposite and adjacent sides of a right triangle)

Output Ports

f - the corresponding angle in radians

Notes

Implemented using Javascript. Will attempt to convert any non-text input value to a decimal string representation before the function is computed.
