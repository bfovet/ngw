NGW Component Documentation - sin

sin

Description

Computes the ratio of the lengths of the opposite side and the hypotenuse of a right triangle from a given angle in radians

Properties

(None)

Input Ports

x - an angle given in radians

Output Ports

f - the ratio of the lengths of the opposite side and the hypotenuse of a right triangle

Notes

Implemented using Javascript. Will attempt to convert any non-text input value to a decimal string representation before the function is computed.
