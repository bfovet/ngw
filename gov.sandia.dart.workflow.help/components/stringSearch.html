NGW Component Documentation - stringSearch

stringSearch

Description

Searches the input string for any instance of the searchValue and returns the index of the string if it exists; if it doesn't exist, outputs a -1

Properties

searchValue - a string to be searched for in the input

Input Ports

x - a string in which the node will search for the searchValue

Output Ports

f - returns the index of the searchValue if it exists within the input; otherwise, it returns a -1

Notes
