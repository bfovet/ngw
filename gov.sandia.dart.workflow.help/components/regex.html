NGW Component Documentation - regex

regex

Description

Attempt to match a regular expression with an optional capture group against some input. Emit the matched data (possibly just the capture group) to one of two outputs.

Properties

regex - the regular expression, with an optional capture group. As an example, if you wanted to match the number in text like -Energy = 101.5- an appropriate regular expession would be -Energy = (.+)"

Input Ports

x - input data to be matched

Output Ports

match - if the input matches the regular expression, then the data - or the capture group, if present - is sent to this output

no_match - if the input doesn-t match the regular expression, the input is sent to this output

Notes

This node uses Java regular expressions as documented here.
