NGW Component Documentation - print

print

Description

Prints formatted output to standard output, and sends the same data to an output port.

Properties

formatString - a Java format string, similar to the one used by C's printf function. See https://docs.oracle.com/javase/tutorial/java/data/numberformat.html.

Input Ports

x - data which becomes the argument that can be included in the formatted String using format conversion.

Output Ports

f - the formatted String is also sent here.

Notes

You can create additional input ports on this node. In that case, the formatting operation will be repeated on each connected input in turn.
