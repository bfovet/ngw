NGW Component Documentation - folder

folder

Description

A node representing the path to a folder on the local filesystem.

Properties

folderName - the name or path of a folder. If this is a folder name or a relative path, it is interpreted relative to the home directory.

Input Ports

folderName - the name or path of a folder. If this is a name or a relative path, it is interpreted relative to the home directory.

Output Ports

folderReference - the absolute path to the folder represented by this node.

Notes

If the folder name is specified using both the property and the input port, the value from the input port takes precedence.
