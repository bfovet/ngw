NGW Component Documentation - compare

compare

Description

Compare two numeric values using a supplied operator, then activate one of two outputs depending on the result.

Properties

operator - a standard Javascript numeric comparison operator ( <, >, ==, etc)

x - the left input (if not provided via an input port)

y - the right input (if not provided via an input port)

Input Ports

x - the left input

y - the right input

Output Ports

true - if the comparison is true, the x input value is provided on this port

false - if the comparison is false, the x input value is provided on this port

Notes

Any Javascript binary operator can be used. Will attempt to convert any non-text input values to decimal string representations before computing the result.
