NGW Component Documentation - getColumns

getColumns

Description

Extracts one or more columns of data from a tabular data source. Columns may be selected either by name or by numerical order.

To select data from columns by name, provide output ports named for each column to be selected. For example, an output port name TIME will be sent data from the column named TIME, etc.

To select data from a (single) column by numerical order, specify the numerical index of the column (starting with 1) via the columnIndex property, and data from the corresponding column will be sent to the selected output port.

Properties

number of header lines
the number of header lines at the beginning of the file before data begins. If named columns are referenced (i.e. via output ports), column names will be expected to be found in the last of the header lines. Optional, defaults to 1.
number of footer lines
the number of non-data lines at the end of the file to be ignored. Optional, defaults to 0.
separator regular expression
a Java regular expression matching the separators between columns. Optional, defaults to "\\s+" (any white space).
columnIndex
the 1-based index (left to right) of a column to send to the column output port. Optional.

Input Ports

data
raw tabular data.
fileName
the name of a file containing tabular data.

Output Ports

column-name
data from the named column

selected
data from the column indexed by the columnIndex property.

Notes

Only one of the two input ports, data and fileName, should be specified.
