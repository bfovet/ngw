NGW Component Documentation - signum

signum

Description

Converts an input to either -1, 0, or 1 if the input is less than 0, equal to 0, or greater than 0, respectively

Properties

(None)

Input Ports

x - a number

Output Ports

f - if x > 0, then f = 1; if x < 0, then f = -1; if x = 0, then f = 0

Notes
