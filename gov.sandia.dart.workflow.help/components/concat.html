NGW Component Documentation - concat

concat

Description

Concatenates all inputs into one string. Ordering is determined alphabetically by input port name, e.g. data found on an input port named x appears in the output string before data found on input port named y, etc.

Properties

(None)

Input Ports

x - a default input (not required)

y - a default input (not required)

Output Ports

f - the concatenated string
