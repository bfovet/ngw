NGW Component Documentation - uploadFile

uploadFile

Description

Uploads a specified file

Properties

hostname - the location the file is to be uploaded to

localFile - the name of the file to be uploaded

remoteFile - optional; the name of the file to be uploaded

remotePath - if uploading a remote file, the working directory to use on the remote machine

Input Ports

trigger - any node; it will initiate the file upload

Output Ports

file - any node

Notes
