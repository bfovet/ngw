NGW Component Documentation - uploadFile

uploadFile

Description

Uploads a specified file

Properties

hostname - the machine the file is to be uploaded to

localFile - the path of the file to be uploaded. If it's a relative path, it is relative to the workflow home.

remoteFile - optional; the name of the file on the remote system.

remotePath - the destination directory to use on the remote machine

Input Ports

localFile - the path of the file to be uploaded. Overrides the property by the same name

Output Ports

file - the resulting name of the remote file.

Notes
