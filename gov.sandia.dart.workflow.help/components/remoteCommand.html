NGW Component Documentation - remoteCommand

remoteCommand

Description

Completes a given command on a remote machine

Properties

hostname - the remote machine where the command is to be completed

command - the task to be completed

remotePath - the working directory on the remote machine

Input Ports

trigger - any node; it will initiate the remote command

Output Ports

output - the result of the remote command

error - any error that was encountered while completing the remote command; can be shown using display
