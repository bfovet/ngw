NGW Component Documentation - fail

fail

Description

Prints a message to stdout and halts the executing workflow by throwing an exception.

Properties

formatString - a Java format string. See https://docs.oracle.com/javase/tutorial/java/data/numberformat.html.

Input Ports

x - triggers the failure. The value can be embedded in the printed output via format conversion.

Output Ports

None.
