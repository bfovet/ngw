NGW Component Documentation - cos

cos

Description

Computes the ratio between the lengths of the adjacent side and hypotenuse of a right triangle from an angle in radians

Properties

(None)

Input Ports

x - an angle in radians

Output Ports

f - the ratio between the lengths of the adjacent side and hypotenuse of a right triangle

Notes

Implemented using Javascript. Will attempt to convert any non-text input value to a decimal string representation before the function is computed.
