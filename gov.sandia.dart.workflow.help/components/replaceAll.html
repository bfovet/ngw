NGW Component Documentation - replaceAll

replaceAll

Description

Replace each substring of x that matches the regular expression regex with the replacement string replacement.

Properties

x
the string on which replacement is to be performed
regex
a Java regular expression to be matched
replacement
the string to be substituted into x for each match of regex

Input Ports

See Properties above.

Output Ports

f
the string after replacement

Notes

See the documentation for the replaceAll method of Java's String class for more detail.
