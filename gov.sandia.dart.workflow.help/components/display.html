NGW Component Documentation - display

display

Description

Creates a dialog box that displays a formatted string to the user with OK and Cancel buttons. If the user presses "Cancel," the workflow is cancelled.

Properties

formatString - a Java format string. See https://docs.oracle.com/javase/tutorial/java/data/numberformat.html.

Input Ports

x - any data; can be incorporated into formatted output.

Output Ports

f - the formatted message.

Notes

Leaving the formatString under Properties empty will cause the display to show the input. However, if there is an input in addition to a formatString and the formatString does not call upon the input, then the formatString takes precedence.
