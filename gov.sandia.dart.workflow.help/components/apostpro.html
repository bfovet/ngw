NGW Component Documentation - apostpro

apostpro

Description

An experimental (and very incomplete) tool for data extraction. Matches a regular expression, skips some number of lines, splits the following line into space or comma-delimited fields, and returns a given field by index.

Properties

inputFile - the input filename to extract data from

match - a Java regular expression. The operation begins by matching this expression.

skip - this number of lines are skipped after matching the expression.

field - the 0-based index of the field to return after skipping the given number of lines.

Optional Properties

separator - a regular expression to be used for splitting the line; if not specified uses [,\s]+ (space or comma delimited)

Input Ports

inputFile - if connected, the file to extract data from

Output Ports

f - the extracted field
